# GGParking

## GyeongGi-do + Parking = GG Parking

GG파킹(GG Parking)은 경기도 지역 주차장 정보를 확인할 수 있도록 돕는 모바일 애플리케이션


<br/>

### 기능
- 내 위치 주변 주차장 확인
- 주차장 검색
- 지역 기반 주차장 검색
- 무료 주차장 필터
- 주차장 상세정보
- 지도 뷰 제공
